🕹️ 控制数据规范 (发往 X板卡)

如果您不使用上述网页键盘，而是通过外部程序投递控制数据到 X板卡，数据链路必须遵循平台统一的 JSON 通信标准。
主要用于同步相机/车体的实时位姿供 AR 追踪渲染，数据扁平化，必须直接包含 pos (三维坐标 XYZ) 与 euler (俯仰/偏航角) 等基础字段，引擎不支持多层嵌套的 JSON 结构。
